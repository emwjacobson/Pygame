# This is my game

## Installation
### Requirements:
- Python3
- Python3 venv

### Steps (Linux)
1. Clone this repo and `cd` into folder
2. `python3 -m venv p3`
3. `source p3/bin/activate`
4. `pip install -r requirements.txt`
5. `python main.py`

### Steps (Windows)
1. Clone this repo and `cd` into folder
2. `python3 -m venv p3`
3. `.\p3\Scripts\activate`
4. `pip install -r requirements.txt`
5. `python main.py`
